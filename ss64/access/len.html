Len

Returns the length of a string.

Syntax
      Len(string)

Key
   string             A string or string expression.

The Len() function can be used in VBA or in an SQL query.

Examples

Dim intDemo as Integer
IntDemo = Len("The world is everlasting")
Returns 24
