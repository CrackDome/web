Abs

Function that returns the absolute value of a number (ignoring positive/negative sign)

Syntax
      Abs (number)


The Abs function will always return a positive number.

Examples

Dim dblDemo

dblDemo = Abs (-123.4)
123.4

dblDemo = Abs (45.1)
45.1
