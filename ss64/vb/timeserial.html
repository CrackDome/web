TimeSerial

Construct a Time value from Hours, Minutes and seconds.

Syntax 
      TimeSerial(Hour, Minute ,Second)

Key
   Hour     A numeric hour (0-23)

   Minute   A numeric minute (0-59)

   Second   A numeric second (0-59)

Example

result = TimeSerial(14,30,07)
WScript.Echo result
> 14:30:07
