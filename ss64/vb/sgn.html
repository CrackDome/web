Sgn

Returns the sign of a number

Syntax
      Sgn(Number)

Key
   Number  The number to return the sign of.

If number > 0, then Sgn() will return 1.
If number = 0, then Sgn() will return 0.
If number < 0, then Sgn() will return -1.

Example

Dim intDemo
intDemo = Sgn(64)
MsgBox intDemo
