IsNumeric

Return TRUE if the expression is a valid number, otherwise return FALSE.

Syntax
      IsNumeric(expression)

Key
   expression    A string or numeric expression.

Examples

Dim boolDemo
boolDemo = IsNumeric(123.45)
Returns True
boolDemo = IsNumeric("123.45")
Returns True
boolDemo = IsNumeric("SS64")
Returns False
