CLIP.exe

Copy the result of any command to the Windows clipboard.

Syntax
      command | CLIP

      CLIP < filename.txt

When using clip.exe in a batch script you should warn the user that their clipboard is about to be overwritten.

Examples:

DIR | CLIP

DATE /t | CLIP

hostname| clip
