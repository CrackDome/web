DROP CREDENTIAL

Remove a credential from the server.

Syntax
      DROP CREDENTIAL credential_name

Requires ALTER ANY CREDENTIAL permission or to drop a system credential, CONTROL SERVER permission.

Example

DROP CREDENTIAL MyCredential;
GO
