insert

insert is a method that will insert one string into another.

Syntax
      .Insert(IntStartIndex, String_value)

 

Example

PS C:\> $text = "Hello World"
PS C:\> $text = $text.Insert(6, "big ")
PS C:\> $text
Hello big World
